# 🇲🇲 Myanmar Alphabet Learning App

A simple, interactive Flutter mobile application designed to help children and beginners learn the Myanmar (Burmese) alphabet through tracing, sound, and visual learning.

---

## 📱 Overview

**Myanmar Alphabet Learning App** is an educational mobile app built with Flutter.  
It helps users learn Myanmar letters through:

- Letter tracing practice
- Audio pronunciation for each character
- Interactive visual feedback and animations
- Child-friendly interface

The goal is to make learning the Myanmar alphabet easy, engaging, and fun.

---

## 🎯 Features

- ✏️ Letter tracing practice with finger input
- 🔊 Audio pronunciation for each Myanmar character
- 🎉 Interactive feedback and animations
- 🧒 Designed for kids and beginners
- 📱 Clean and simple UI optimized for mobile devices
- 📢 Integrated ads (Google AdMob)

---

## 🛠 Tech Stack

- Flutter (Dart)
- google_mobile_ads (AdMob)
- just_audio (audio playback)
- confetti (UI effects)
